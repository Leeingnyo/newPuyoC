# new Puyo C

![Play GIF](./newPuyoC.gif)

new Puyo C with C++ on cross platform (windows & POSIX), which can play 1 vs 1 remote battle

## How to Build

```
git clone https://github.com/Leeingnyo/newPuyoC.git
cd newPuyoC
make
./new_puyo_c
```
